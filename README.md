## 如何使用

### 安装
1. 阅读[《免责声明》](https://github.com/SRInternet-Studio/Wallpaper-generator/blob/NEXT-PREVIEW/DISCLAIMER.md)
2. 下载 [Releases](https://github.com/SRInternet-studio/Wallpaper-generator/releases) 中的最新的 **Release** 安装程序（.exe）；
3. 双击安装，跟着提示一步步来；
4. 启动软件。

### 生成壁纸
1. 在软件主界面顶部的导航栏内可以切换不同的图片类型，以进入不同的生成模式；
2. 如果没有可以生成的图片类型或没有自己喜欢的类型，可以切到**图片源市场**标签页，添加自己喜欢的图片类型，然后重启壁纸生成器；
3. 首次生成建议前往**设置**标签页，将图片保存的路径选择到自己喜欢的位置，避免过多占用系统空间；
4. 回到生成页面，调整生成数量和类型等各种参数，点击生成；
5. 等待提示生成结束，就能在右侧看到生成的图片；
6. 点击右侧已经生成的图片可以打开下浮菜单，支持设置为壁纸、通过系统图片查看器打开、通过资源管理器打开等功能和信息的查询；
7. 点击右下角的页面指示器可以来回滚动图片哦

### 如何开机自启和自动更换
1. 在软件**欢迎**标签页内点击【自动更换壁纸的托盘程序】；
2. 可以在**自动更换壁纸**窗口内启用自动更换，然后设置更换间隔、是否开机自启等；
3. 在成功启用自动更换壁纸后，您会在窗口中看到距离下一次自动更换的倒计时；
4. 在任务栏右下方托盘图标中，您可以右键并点击【立即更换壁纸】以跳过本轮倒计时直接更换壁纸（若没有看到则可能被隐藏起来了，请点击托盘处最左侧的向上的箭头的图标 *^* 以展开被隐藏的托盘图标）；
5. 点击【应用】即可保存设置，关闭窗口即可回到欢迎页面。

## 如何部署源代码
> 壁纸生成器 5.0.0 绝大部分采用了 Python [^1]，这要求你的电脑中需要包含 Python>=3.10 的标准环境和所需环境 [(requirements.txt)](requirements.txt)

### 配置所需 Python 环境
1. 前往[官网](https://www.python.org)下载 Python 3.10 及以上版本，推荐 Python 3.12；
2. 根据安装程序的提示安装 Python；
3. 新建 Python 虚拟环境
   ```bash
   python -m venv wpg_next
   ```
4. 下载所需的第三方库；
   ```bash
   python -m pip install -r requirements.txt
   ```
5. 启动程序 `MainWindow.py`
   ```bash
   python ./MainWindow.py
   ```
